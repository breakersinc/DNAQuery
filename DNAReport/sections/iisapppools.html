Cuentas IIS Application Pool

Las cuentas de IIS Application Pool son cuentas de servicio que realizan funciones en nombre de una aplicación web IIS. Al igual que con otras cuentas de servicio, estas deben administrarse con cuidado, ya que generalmente tienen permisos de acceso para archivos de aplicaciones y fuentes de datos locales que pueden ser objetivos para los atacantes. Para algunas aplicaciones, estas cuentas también deben tener acceso a almacenes de datos más confidenciales, como instancias de bases de datos o incluso otras aplicaciones. A menudo, estas son cuentas de dominio en lugar de cuentas locales para facilitar la administración, pero independientemente del tipo de cuenta, deben administrarse de manera efectiva para asegurar el entorno de manera adecuada. Esto significa que el acceso a estas credenciales debe ser limitado y, cuando sea posible, las contraseñas de estas cuentas deben cumplir con las políticas de la organización que rigen la edad y la frecuencia de rotación.
